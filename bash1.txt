1. Открыть домашнюю директорию через терминал - cd

2. Определить имя папки, в которой вы находитесь - pwd

3. Создать внутри этой папки каталог с именем test1 - mkdir test1

4. Перейти в папку test1 - cd test1

5. Создать файл 1,2 и 3 внутри каталога test1 - touch 1.txt 2.txt 3.txt

6. Проверить содержимое каталога test1 - ls

7. Перейти в домашнюю директорию - cd

8. Создать папку test2 внутри домашней директории - mkdir test2

9. Удалить папку test2 - rmdir test2

10. Удалить файл 2 из папки test1 - rm test1/2.txt

11. Создать папку в домашней директории test3 и добавить в нее два файла - mkdir test3 => touch test3/file1.txt test3/file2.txt

12. Удалить папку test3 - rm -r test3

13. Создать папку test4 в домашней директории - mkdir test4

14. Переместить файлы 1 и 3 из папки test1 в папку test4 - mv test1/1.txt test1/3.txt test4

15. Добавить в файл 1 три строки со словами line - echo line > test4/1.txt => echo line >> test4/1.txt => echo line >> test4/1.txt

16. Посмотреть содержимое файла 1 - cat test4/1.txt

17. Добавьте в файл 3 три строки со словами line - echo line > test4/3.txt => echo line >> test4/3.txt => echo line >> test4/3.txt

18. Просмотрите содержимое двух файлов (1 и 3) сразу - cat test4/1.txt test4/3.txt

19. Используя один из редакторов замените все строки в файле 1 - nano test4/1.txt => заменить строки вручную, ctrs+s - сохранить, ctrl+x - выйти
